# Lista em Pilha Dinâmica de Números Inteiros.

Um trabalho que fiz da matéria de Estruturas de Dados I do curso de Sistema de Informação da UFPI – Picos –Piauí – Brasil.

É uma lista em pilha dinâmica que recebe números inteiros em linguagem C. O programa cria uma lista que armazena números inteiros em forma de pilha.
Ele cria a lista, insere os números, busca os números, mostra o topo da pilha, remover números da lista, mostra os elementos da lista e exclui a lista liberando-a.
